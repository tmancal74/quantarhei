Quantarhei's Dependencies
=========================

Quantarhei depends on a number of external tools. To install Quantarhei
you need the following Python packages.

- `numpy`_  
- `scipy`_
- `pytorch`_
- `matplotlib`_
- `terminaltables`_
- `h5py`_

To develop Quantarhei, you need the following packages

- `paver`_
- `nose`_
- `aloe`_
- `coverage`_
- `codecov`_
